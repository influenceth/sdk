# Raw Image Inputs

This directory stores raw image inputs for the generated image payload. It is organized type-first so related source art stays together.

- `asteroids/`: asteroid card/compositor source art.
- `buildings/`: building icons, site icons, sprite sheets, and atlases.
- `crewmates/`: crewmate compositor layers, sprite sheets, and atlases.
- `resources/`: resource icons, sprite sheets, and atlases.
- `ships/`: ship icons, holo icons, lot icon overrides, compositor art, sprite sheets, and atlases.
- `shared/card/`: shared card/compositor layers.
- `sway/icons/`: Sway brand icon source files.
